HTVG provides four fundamental element types that form the building blocks of any document. Each element type has a specific role and set of style properties.

## Box Element

The `box` element is a basic block container. It's the simplest container type, ideal for simple layouts without flex properties.

### Schema

```typescript
{
  type: "box",
  style?: BoxStyle,
  children?: Element[]
}
```

### Example

<Tabs>
  <Tab title="Basic Box">
    ```json
    {
      "type": "box",
      "style": {
        "width": 200,
        "height": 100,
        "backgroundColor": "#2563eb",
        "borderRadius": 8
      },
      "children": []
    }
    ```
  </Tab>
  <Tab title="With Children">
    ```json
    {
      "type": "box",
      "style": {
        "padding": 20,
        "backgroundColor": "#ffffff",
        "borderWidth": 1,
        "borderColor": "#e0e0e0"
      },
      "children": [
        {
          "type": "text",
          "content": "Box content",
          "style": { "fontSize": 16 }
        }
      ]
    }
    ```
  </Tab>
</Tabs>

### Box Style Properties

<ParamField path="display" type="string" default="block">
  Display mode: `"block"`, `"flex"`, or `"none"`
</ParamField>

<ParamField path="width" type="number | string">
  Width in pixels (number) or percentage (string like `"50%"`)
</ParamField>

<ParamField path="height" type="number | string">
  Height in pixels (number) or percentage (string like `"50%"`)
</ParamField>

<ParamField path="minWidth" type="number | string">
  Minimum width constraint
</ParamField>

<ParamField path="maxWidth" type="number | string">
  Maximum width constraint
</ParamField>

<ParamField path="minHeight" type="number | string">
  Minimum height constraint
</ParamField>

<ParamField path="maxHeight" type="number | string">
  Maximum height constraint
</ParamField>

<ParamField path="margin" type="number | string" default="0">
  Margin spacing. Can be:
  - Single number: `20` (all sides)
  - Space-separated: `"10 20"` (vertical horizontal)
  - Four values: `"10 20 30 40"` (top right bottom left)
</ParamField>

<ParamField path="padding" type="number | string" default="0">
  Padding spacing with same syntax as margin
</ParamField>

<ParamField path="backgroundColor" type="string">
  Background color (hex, rgb, rgba, or named color)
</ParamField>

<ParamField path="borderWidth" type="number" default="0">
  Border width in pixels
</ParamField>

<ParamField path="borderColor" type="string">
  Border color
</ParamField>

<ParamField path="borderRadius" type="number | string" default="0">
  Corner radius. Can be:
  - Single number: `8` (all corners)
  - Four values: `"8 8 0 0"` (top-left top-right bottom-right bottom-left)
</ParamField>

<ParamField path="opacity" type="number" default="1">
  Opacity from 0 (transparent) to 1 (opaque)
</ParamField>

## Flex Element

The `flex` element is a flexbox container supporting full CSS Flexbox layout. It's the most powerful container type for complex layouts.

### Schema

```typescript
{
  type: "flex",
  style?: FlexStyle,
  children?: Element[]
}
```

### Example

<Tabs>
  <Tab title="Column Layout">
    ```json
    {
      "type": "flex",
      "style": {
        "flexDirection": "column",
        "gap": 12,
        "padding": 20,
        "backgroundColor": "#f8f9fa"
      },
      "children": [
        {
          "type": "text",
          "content": "Item 1",
          "style": { "fontSize": 16 }
        },
        {
          "type": "text",
          "content": "Item 2",
          "style": { "fontSize": 16 }
        }
      ]
    }
    ```
  </Tab>
  <Tab title="Centered Content">
    ```json
    {
      "type": "flex",
      "style": {
        "flexDirection": "column",
        "justifyContent": "center",
        "alignItems": "center",
        "width": 400,
        "height": 300,
        "backgroundColor": "#ffffff"
      },
      "children": [
        {
          "type": "text",
          "content": "Centered Text",
          "style": { "fontSize": 24, "fontWeight": "bold" }
        }
      ]
    }
    ```
  </Tab>
  <Tab title="Badge Row">
    ```json
    {
      "type": "flex",
      "style": {
        "flexDirection": "row",
        "gap": 10,
        "flexWrap": "wrap"
      },
      "children": [
        {
          "type": "flex",
          "style": {
            "backgroundColor": "#22c55e",
            "borderRadius": 16,
            "padding": "6 14"
          },
          "children": [
            {
              "type": "text",
              "content": "Passing",
              "style": { "fontSize": 13, "fontWeight": 600, "color": "#ffffff" }
            }
          ]
        }
      ]
    }
    ```
  </Tab>
</Tabs>

### Flex Style Properties

Flex elements inherit all Box style properties, plus additional flex-specific properties:

<ParamField path="flexDirection" type="string" default="row">
  Flex direction: `"row"`, `"column"`, `"row-reverse"`, `"column-reverse"`
</ParamField>

<ParamField path="justifyContent" type="string" default="flex-start">
  Main axis alignment:
  - `"flex-start"` - Align to start
  - `"flex-end"` - Align to end
  - `"center"` - Center items
  - `"space-between"` - Space between items
  - `"space-around"` - Space around items
  - `"space-evenly"` - Even spacing
</ParamField>

<ParamField path="alignItems" type="string" default="stretch">
  Cross axis alignment:
  - `"flex-start"` - Align to start
  - `"flex-end"` - Align to end
  - `"center"` - Center items
  - `"stretch"` - Stretch to fill
  - `"baseline"` - Align baselines
</ParamField>

<ParamField path="gap" type="number" default="0">
  Gap between flex items in pixels
</ParamField>

<ParamField path="flexWrap" type="string" default="nowrap">
  Wrapping behavior: `"nowrap"` or `"wrap"`
</ParamField>

<Note>
  All dimension properties (`width`, `height`, `margin`, `padding`, `backgroundColor`, `borderWidth`, `borderColor`, `borderRadius`, `opacity`) work the same as in Box elements.
</Note>

## Text Element

The `text` element is a leaf element for rendering text with automatic line wrapping and proper text shaping via Parley.

### Schema

```typescript
{
  type: "text",
  content: string,
  style?: TextStyle
}
```

### Example

<Tabs>
  <Tab title="Basic Text">
    ```json
    {
      "type": "text",
      "content": "Hello, HTVG!",
      "style": {
        "fontSize": 24,
        "fontWeight": "bold",
        "color": "#1a1a1a"
      }
    }
    ```
  </Tab>
  <Tab title="Paragraph">
    ```json
    {
      "type": "text",
      "content": "This is a self-contained document rendered to SVG. Text automatically wraps based on the available width.",
      "style": {
        "fontSize": 16,
        "lineHeight": 1.5,
        "color": "#666666"
      }
    }
    ```
  </Tab>
  <Tab title="Centered">
    ```json
    {
      "type": "text",
      "content": "Centered Text",
      "style": {
        "fontSize": 20,
        "fontWeight": 700,
        "textAlign": "center",
        "color": "#1a1a1a"
      }
    }
    ```
  </Tab>
</Tabs>

### Text Style Properties

<ParamField path="fontFamily" type="string" default="sans-serif">
  Font family name
</ParamField>

<ParamField path="fontSize" type="number" default="16">
  Font size in pixels
</ParamField>

<ParamField path="fontWeight" type="number | string" default="400">
  Font weight: number (100-900) or keyword (`"normal"` = 400, `"bold"` = 700)
</ParamField>

<ParamField path="lineHeight" type="number" default="1.2">
  Line height as a multiplier of font size
</ParamField>

<ParamField path="textAlign" type="string" default="left">
  Text alignment: `"left"`, `"center"`, `"right"`, `"justify"`
</ParamField>

<ParamField path="color" type="string" default="#000000">
  Text color (hex, rgb, rgba, or named)
</ParamField>

<ParamField path="letterSpacing" type="number" default="0">
  Letter spacing in pixels
</ParamField>

<ParamField path="textRendering" type="string" default="text">
  Rendering mode:
  - `"text"` - Render as SVG `<text>` element
  - `"vector"` - Render as vector paths (future feature)
</ParamField>

<ParamField path="flexGrow" type="number" default="0">
  Flex grow factor when inside a flex container
</ParamField>

<ParamField path="flexShrink" type="number" default="1">
  Flex shrink factor when inside a flex container
</ParamField>

<Tip>
  Text elements automatically wrap based on the available width calculated by the layout engine. You don't need to manually insert line breaks.
</Tip>

## Image Element

The `image` element renders images with intrinsic dimensions. Images must provide width and height.

### Schema

```typescript
{
  type: "image",
  src: string,
  width: number,
  height: number,
  style?: ImageStyle
}
```

### Example

<Tabs>
  <Tab title="Basic Image">
    ```json
    {
      "type": "image",
      "src": "data:image/png;base64,iVBORw0KGg...",
      "width": 100,
      "height": 100,
      "style": {}
    }
    ```
  </Tab>
  <Tab title="Rounded Image">
    ```json
    {
      "type": "image",
      "src": "https://example.com/image.png",
      "width": 200,
      "height": 150,
      "style": {
        "borderRadius": 12,
        "objectFit": "cover"
      }
    }
    ```
  </Tab>
  <Tab title="Constrained">
    ```json
    {
      "type": "image",
      "src": "data:image/jpeg;base64,...",
      "width": 800,
      "height": 600,
      "style": {
        "maxWidth": 400,
        "objectFit": "contain"
      }
    }
    ```
  </Tab>
</Tabs>

### Image Style Properties

<ParamField path="width" type="number | string">
  Override intrinsic width
</ParamField>

<ParamField path="height" type="number | string">
  Override intrinsic height
</ParamField>

<ParamField path="minWidth" type="number | string">
  Minimum width constraint
</ParamField>

<ParamField path="maxWidth" type="number | string">
  Maximum width constraint
</ParamField>

<ParamField path="minHeight" type="number | string">
  Minimum height constraint
</ParamField>

<ParamField path="maxHeight" type="number | string">
  Maximum height constraint
</ParamField>

<ParamField path="margin" type="number | string" default="0">
  Margin spacing (same syntax as Box)
</ParamField>

<ParamField path="objectFit" type="string" default="contain">
  Image fit mode:
  - `"contain"` - Scale to fit, preserving aspect ratio
  - `"cover"` - Scale to cover, preserving aspect ratio
  - `"fill"` - Stretch to fill
</ParamField>

<ParamField path="borderRadius" type="number | string" default="0">
  Corner radius for clipping
</ParamField>

<ParamField path="opacity" type="number" default="1">
  Image opacity (0-1)
</ParamField>

<ParamField path="flexGrow" type="number" default="0">
  Flex grow factor when inside a flex container
</ParamField>

<ParamField path="flexShrink" type="number" default="1">
  Flex shrink factor when inside a flex container
</ParamField>

<Info>
  The `src` field accepts both data URIs (`data:image/png;base64,...`) and external URLs. For self-contained SVG output, use data URIs.
</Info>

## Type Definitions

The complete type definitions are available in the source code at `crates/htvg-core/src/element.rs`. Here's the core enum:

```rust
pub enum Element {
    Box {
        style: BoxStyle,
        children: Vec<Element>,
    },
    Flex {
        style: FlexStyle,
        children: Vec<Element>,
    },
    Text {
        content: String,
        style: TextStyle,
    },
    Image {
        src: String,
        width: f32,
        height: f32,
        style: ImageStyle,
    },
}
```
